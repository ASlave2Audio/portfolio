Professionally

I describe myself as "T shaped" UX resource. My skill set is broad and deep in hypotheses (in)validation, prototyping and usability. I enjoy tackling products in rapid paced agile teams and have lead several design efforts from concept to completion. I strive to be the voice of the user in all stages of the design process, and ensure that data and empathy is the driving force behind every minute decision.

Process

Observe

Collecting data on user needs and goals is a vital step in ensuring a fruitful design. This is done by:

  • Contextual Inquiries
  • Storyboarding
  • Surveying
  • Card Sorting

Iterate

I focus on creating low-fidelity iterations in order to hash out preliminary design issues and to better visualize my ideas to the client. This is done by:

  • Whiteboarding
  • Wireframing
  • Creating Page Flows
  • Prototyping

Refine

Once a general design is established, I continue to refine and test the interface to ensure the user's needs are effectively met. This is done by:

  • User Testing
  • Creating Photoshop Comps
  • Developing
